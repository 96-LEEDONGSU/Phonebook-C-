# Phonebook-C-
C언어를 이용한 전화번호부

해당 코드는 <소프트웨어공학> 수업에서 전화번호부 과제를 위해 제작하였다.   
프로그램을 만드는 과정은 요구분석-설계-구현-테스트 과정을 거쳐 만들어진다.    
가장 기초적으로 요구분석과 설계를 간접적으로 간단하게 경험하기 위해 이미 구조를 충분히 파악하고 있는 전화번호부를 제작하였다.
